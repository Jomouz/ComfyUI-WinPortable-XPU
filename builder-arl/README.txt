This folder uses a special PyTorch and IPEX 
for Intel Arrow Lake-H (Ultra Series 2 Mobile Processors).

This folder won't get much update as the 'builder' folder, 
since the 'builder' uses XPU PyTorch that fits most modern Intel GPU models.

Ref:
https://pytorch-extension.intel.com/?platform=gpu&version=v2.6.10%2Bxpu&os=windows&package=pip
